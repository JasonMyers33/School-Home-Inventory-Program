# School-Home-Inventory-Program

ITS Python Project
Program Name: Home Inventory Program
Program Author: Jason Myers
Program Objective: Create a home inventory class that will be used by a National Builder to maintain inventory of available houses in the country.  

Pseudocode:
Create a dictionary for the homes
Private fuctions for:
Square Feet
Address
City
State
Zip Code
Model Name
Sale Status only options are:sold, available, under contract
Methods to include:
constructor
add a new home
remove a home
update home attributes
view inventory on screen
output all home inventory to a CSV file.

Program Inputs: dictionary, home parts - address, squarefeet, city, state, zip code, model name, sale status
Program Outputs: Print dict. export to CSV.
